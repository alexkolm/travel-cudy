# travel-cudy

OpenWrt router automation for dynamic tunnel switching via WireGuard and VK TURN.

## Documentation

- 🇷🇺 Russian documentation: [README.ru.md](README.ru.md)

## Features

- Tunnel / direct mode switching
- WireGuard integration
- Hotplug automation
- LED status indication
